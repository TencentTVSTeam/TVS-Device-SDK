◆ aisdkRecvSendApListByJson()

AISDK_API_EXPORTS int aisdkRecvSendApListByJson ( const char *  apListJson,
int  length 
)

将附件的wifi热点信息以json形式发送给SDK

注解
当收到CMD:AISDK_CMD_RECV_GET_AP_LIST时应该调用此函数或者aisdkRecvSendApList
参数
apListJsonwifi热点json格式。
示例
{"res":"1",
"aplist":[
{"ssid":"wmmAudio_a7b8",
"bssid":"00:22:6c:00:a7:b8",
"rssi":"76",
"channel":"1",
"auth":"OPEN",
"encry":"NONE",
"extch":"1"
}]
}
数据说明
字段说明
res返回列表数量
aplistssid列表
ssidWifi的可见名称, 应该用十六进制编码。
bssidMac地址
rssi信号强度，正整数形式。
channelWifi的信道
auth是否加密。OPEN表示未加密，WPA2PSK表示加密
encry加密类型。AES或者NONE
lengthapListJson字符串长度
返回
0:成功 其他失败

◆ aisdkRecvSendDeviceInfo()

AISDK_API_EXPORTS int aisdkRecvSendDeviceInfo ( const char *  deviceSerialNum,
const char *  productId,
const char *  apName,
const char *  netCard,
const char *  guid 
)

向SDK传递设备信息。

注解
当收到CMD:AISDK_CMD_RECV_GET_INFO时应该调用此函数
参数
deviceSerialNum设备序列号，应能唯一识别一台设备
productId产品ID。叮当平台分配
apName出厂设备开启的AP SSID。
netCard网卡名，如果网卡没有特殊配置，传NULL即可，默认是wlan0。
guid默认传NULL即可。
返回
0表示成功，其他表示失败
