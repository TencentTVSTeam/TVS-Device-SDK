Introduction

通用语音识别相关接口，只有一路音频输入的情况使用这个接口即可完成唤醒识别和语音识别流程 Created by Chuancong Zhu on 2018/9/21. Copyright © 2018年 Zacard Fang. All rights reserved.


Protocols

OneShotSessionDelegate

OneShot回调接口，在OneshotSession对象中注入


Methods

+cancelOneShotOnlineVoice2Text

取消本次语音识别

+init:wakeupModelPath:

OneShot初始化，只需要初始化一次。

+inputVoice2TextAudioData:

输入音频数据，在OneShot流程中全程需要输入

+start:voiceType:

开始通用语音识别流程，启动一次即可一直输入语音，不需要反复启动，SDK内部会在唤醒识别和语音识别之间切换

+startOneShotOnlineVoice2Text:voiceType:

开始一次语音识别，用于直接唤醒开始交互

+stopOneShot

取消通用语音识别流程，会停掉正在进行中的唤醒和者语音识别

+stopOneShotOnlineVoice2Text

停止本次语音识别并获取结果，用于长按开启语音识别的场景

cancelOneShotOnlineVoice2Text

取消本次语音识别 

-(int)cancelOneShotOnlineVoice2Text;
Return Value

0：ok，other：fail。 错误码定义见K_AISDK_ERROR_*常量

init:wakeupModelPath:

OneShot初始化，只需要初始化一次。 

-(instancetype)init:(NSString*)voiceModelPath 
        wakeupModelPath:(NSString*)wakeupModelPath;
Parameters
voiceModelPath

本地VAD模型所在路径，如果不需要本地VAD可以传NULL

wakeupModelPath

唤醒词模型所在路径，必传，否则无法唤醒

inputVoice2TextAudioData:

输入音频数据，在OneShot流程中全程需要输入 

-(int)inputVoice2TextAudioData:(NSData *)audioBuffer;
Parameters
audioBuffer

录音数据存储区域。建议长度为4096.

Return Value

0：ok，other：fail。 错误码定义见K_AISDK_ERROR_*常量

start:voiceType:

开始通用语音识别流程，启动一次即可一直输入语音，不需要反复启动，SDK内部会在唤醒识别和语音识别之间切换 

-(int)start:(void*)userData voiceType:(NSInteger)flags;
Parameters
userData

自定义数据指针。callback时带回。

flags

控制标志，参考AISDK_FLAG_*常量定义，支持多flag或运算。如果不设置，传0即可。

Return Value

0：ok，other：fail。 错误码定义见K_AISDK_ERROR_*常量

startOneShotOnlineVoice2Text:voiceType:

开始一次语音识别，用于直接唤醒开始交互 

-(int)startOneShotOnlineVoice2Text:(void*)userData voiceType:(NSInteger)flags;
Parameters
userData

自定义数据指针。callback时带回。

flags

控制标志，参考AISDK_FLAG_*常量定义，支持多flag或运算。如果不设置，传0即可。

Return Value

0：ok，other：fail。 错误码定义见AISDK_ERROR_*常量

stopOneShot

取消通用语音识别流程，会停掉正在进行中的唤醒和者语音识别 

-(int)stopOneShot;
Return Value

0：ok，other：fail。 错误码定义见K_AISDK_ERROR_*常量

stopOneShotOnlineVoice2Text

停止本次语音识别并获取结果，用于长按开启语音识别的场景 

-(int)stopOneShotOnlineVoice2Text;
Return Value

0：ok，other：fail。 错误码定义见K_AISDK_ERROR_*常量
