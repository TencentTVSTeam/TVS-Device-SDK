Introduction

文本转语音能力提供者，该类封装了文本转语音能力的所有接口


Protocols

TtsSessionDelegate

文本转语音回调接口 TtsSessionDelegate 会在TtsSession对象中注入


Methods

+cancelText2Speech

取消此次的TTS请求

WARNING:

取消后， 这一次的tts请求结果不再返回。

+text2Speech:userData:

TTS识别接口， TTS结果通过回调异步回调返回

+text2SpeechEx:text:userData:

TTS转换接口， TTS结果通过回调异步回调返回

cancelText2Speech

取消此次的TTS请求

WARNING:

取消后， 这一次的tts请求结果不再返回。 

-(int)cancelText2Speech;
Return Value

0:ok other:fail。 错误码定义见K_AISDK_ERROR_*常量

text2Speech:userData:

TTS识别接口， TTS结果通过回调异步回调返回 

-(int)text2Speech:(NSString *)text userData:(id)userdata;
Parameters
text

需要转语音的文本，以utf8格式编码的文本串，以'\0'结尾

userdata

自定义数据

Return Value

0:ok, other：fail。 错误码定义见K_AISDK_ERROR_*常量

text2SpeechEx:text:userData:

TTS转换接口， TTS结果通过回调异步回调返回 

-(int)text2SpeechEx:(int)languageType text:(NSString *)text 
        userData:(id)userdata;
Parameters
languageType

语言类型，0：默认，1：中文，2：英文

text

需要转语音的文本，以utf8格式编码的文本串，以'\0'结尾

userdata

自定义数据

Return Value

0:ok, other：fail。 错误码定义见K_AISDK_ERROR_*常量

Constants

K_AISDK_CMD_TTS_ERROR

回调接口命令定义，当TTS返回出错时发出

K_AISDK_CMD_TTS_RESULT

回调接口命令定义，当TTS返回结果时发出

K_AISDK_CONFIG_TTS_ENV_TYPE

配置TTS服务的环境

设置TTS环境为正式环境 aisdkSetConfig(AISDK_CONFIG_TTS_ENV_TYPE,AISDK_CONFIG_VALUE_ENV_TYPE_FORMAL):

K_AISDK_RESULT_CODE_TTS_DATA

TTS的语音包返回code，标识中间语音数据包

K_AISDK_RESULT_CODE_TTS_LAST_DATA

TTS的最后一个语音包返回code，标识最后一个语音数据包。

Note:

与AISDK_RESULT_CODE_TTS_DATA的不同在于:当文本较短时，tts语音包较短，只会有AISDK_RESULT_CODE_TTS_LAST_DATA
